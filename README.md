# QR Code Generator

A simple and responsive QR Code Generator built with **HTML**, **CSS**, **Bootstrap**, and **JavaScript**. This tool allows users to quickly generate QR codes for text, URLs, or any other input, making sharing information easier than ever.

## Features

- **User-Friendly Interface:** Intuitive and responsive design using Bootstrap.
- **Real-Time QR Code Generation:** Instant QR code creation as you type.
- **Customizable Inputs:** Supports text, URLs, and more.
- **Lightweight and Fast:** Minimalist codebase for quick performance.
- **Free and Open Source:** Freely distributed for personal and commercial use.


## How to Use

1. Enter the text or URL in the input field.
2. Click the "Generate" button.
3. Download or share the QR code instantly.

## Technologies Used

- **HTML**: For structure and content.
- **CSS**: For custom styling and layout adjustments.
- **Bootstrap**: For responsive design and UI components.
- **JavaScript**: For functionality and QR code generation logic.
